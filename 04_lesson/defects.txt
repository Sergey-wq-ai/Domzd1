1. КРИТИЧЕСКИЙ: Неправильная работа capitilize() с пробелами
ID: SU-001
Метод: capitilize()
Приоритет: Высокий
Статус: Открыт
Описание: Метод не делает первую букву заглавной, если строка начинается с пробела
Шаги воспроизведения:
  1. Вызвать utils.capitilize(" skypro")
  2. Сравнить результат с ожидаемым
Ожидаемый результат: " Skypro"
Фактический результат: " skypro"
Влияние: Некорректное форматирование строк с начальными пробелами

2. СРЕДНИЙ: Неэффективная реализация trim()
ID: SU-002
Метод: trim()
Приоритет: Средний
Статус: Открыт
Описание: Использование цикла while для удаления пробелов
Проблема: Неэффективно для строк с большим количеством пробелов
Рекомендация: Использовать встроенный метод lstrip()
